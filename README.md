# 🐱 健康情緒貓咪陪伴系統

結合健檢數據和心理測驗，生成個人化貓咪陪伴圖片與AI回饋的Streamlit網站系統。

## 📋 專案概述
- 健康檢查數據分析與輸入
- 心理情緒狀態測驗 (Emo Guide)
- AI生成個人化貓咪圖片（橘貓、藍貓、賓士貓等）
- 智能文字回饋系統

## 🛠 技術棧
- **前端**: Streamlit
- **後端**: Python
- **資料庫**: 待定
- **部署**: Streamlit Cloud 或 Google Cloud Run
- **AI**: 圖像/文字生成API
